


## Installation Steps 


**1. Go into the project directory**  <br />
   ``` cd Realtor ``` 
   <br />
   
**2. Install all the dependencies** <br />
    ``` npm i ``` 
    <br />
    
**3. Start the application development server**<br />
    ``` npm start ```

<br /> 


**The app is now running**
<br />
